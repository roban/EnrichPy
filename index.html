Download and Install

Email me with any questions about downloading and installation.

The easiest way to download and install EnrichPy is to use easy_install or pip.

Install with easy_install

You can install the latest release from PyPI by running: 

$ sudo easy_install EnrichPy

Install with pip

You can install the latest release from PyPI by running: 

$ sudo pip install EnrichPy
