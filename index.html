Download and Install

Email me with any questions about downloading and installation.

The easiest way to download and install EnrichPy is to use easy_install.

Install with easy_install (recommended)

Latest release

You can install the latest release from PyPI by running: 

$ easy_install --upgrade -z EnrichPy

Note that the -z option is crucial, otherwise the yield data files will not be installed properly (see this stackoverflow question).
